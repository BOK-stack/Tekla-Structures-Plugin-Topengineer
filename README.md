# Tekla Structures Plugin - Topengineer

Комплексный плагин для Tekla Structures (версии 2020-2025) с функциональностью автоматизации проектирования металлоконструкций в соответствии с требованиями ГОСТ.

## Возможности

### 1. Работа с каталогами
- ✅ Каталоги болтов и крепежа по ГОСТ
- ✅ Каталоги профилей по ГОСТ (ДВУТАВР, швеллер, уголок, труба и т.д.)
- ✅ Тонкостенные Z и C профили
- ✅ Создание нестандартных метизов (анкерные болты, шпильки)
- ✅ Листовая сталь по ГОСТ
- ✅ Материалы с обозначениями русскими буквами

### 2. Генерация ведомостей и отчетов
- ✅ Выгруз��а в Excel
- ✅ Технические спецификации (КМ)
- ✅ Ведомость расхода стали (КЖ)
- ✅ Ведомость сварных швов
- ✅ Нестандартные метизы в ведомостях
- ✅ Карты раскроя проката

### 3. Автоматизация сварки
- ✅ Автоматическая расстановка сварки в модели
- ✅ Подбор размера катета шва
- ✅ Отображение сварки штрихами и крестиками на чертеже
- ✅ Ручная доработка прорисовки сварки

### 4. Управление атрибутами и префиксами
- ✅ Управление префиксами из единого меню
- ✅ Указание нагрузок для КМ
- ✅ Массовый перенос атрибутов между полями
- ✅ Сравнение сборок и определение различий

### 5. Автоматизация размеров и линий
- ✅ Разрезы по ГОСТ (обозначение цифрами)
- ✅ Обозначение линий разрыва по ГОСТ
- ✅ Размеры по треугольнику для ферм
- ✅ Ав��оматическое округление размеров до 1 мм
- ✅ Высотные отметки из мм в метры

### 6. Работа с чертежами
- ✅ Титульные листы автоматически (любой размер, ориентация)
- ✅ Шаблоны для КМ, КМД, КЖ, АР
- ✅ Универсальные настройки печати
- ✅ Предустановки для каждого типа чертежа
- ✅ Компоновки деталей (А4 до А0)
- ✅ Сборочный план здания с указанием местоположения

### 7. Дополнительные функции
- ✅ Подбор длин болтов по ГОСТ
- ✅ Преобразование профилей (прокатные в сварные)
- ✅ Выгрузка данных в SQL/1С
- ✅ Массовая нумерация чертежей
- ✅ Мультивыноски с переменными
- ✅ Вставка таблиц из Excel
- ✅ Автоматическое создание бэкапов

## Требования

- **Tekla Structures**: 2020 SP1 и выше (до 2025)
- **.NET Framework**: 4.7.2 и выше
- **Visual Studio**: 2019 или выше (для разр��ботки)

## Установка

1. Скачайте последний релиз из [Releases](../../releases)
2. Распакуйте файлы в папку плагинов Tekla Structures:
   - Windows: `C:\ProgramData\Trimble\Tekla Structures\<версия>\Extensions\Applications`
3. Перезагрузите Tekla Structures
4. Плагин появится в меню **Applications** → **Topengineer**

## Использование

### Быстрый старт

1. Откройте модель в Tekla Structures
2. Перейдите в **Applications** → **Topengineer**
3. Выберите нужный модуль:
   - **Catalogs** - работа с каталогами
   - **Reports** - генерация ведомостей
   - **Welding** - автоматизация сварки
   - **Drawing** - параметры чертежей
   - **Attributes** - управление атрибутами

### Примеры

Смотрите папку `samples/` для готовых примеров использования.
