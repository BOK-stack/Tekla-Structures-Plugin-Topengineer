# Changelog

## [1.0.0] - 2026-07-20

### Added
- ✅ Комплексная система управления каталогами (болты, профили, материалы)
- ✅ Автоматическая генерация технических спецификаций (КМ)
- ✅ Генерация ведомостей расхода стали (КЖ)
- ✅ Ведомость сварных швов
- ✅ Карты раскроя проката
- ✅ Автоматизация сварки (расстановка и подбор размера катета)
- ✅ Управление атрибутами и префиксами
- ✅ Сравнение сборок
- ✅ Автоматизация чертежей (округление размеров, конверсия отметок)
- ✅ Экспорт в Excel и CSV
- ✅ Система логирования
- ✅ Поддержка ГОСТ стандартов
- ✅ Полная документация на русском языке
- ✅ API Reference для разработчиков

### Technical Details
- Построено на .NET Framework 4.7.2
- Совместимо с Tekla Structures 2020-2025
- Модульная архитектура
- Полная поддержка русского языка и обозначений

### Documentation
- Руководство по установке
- Руководство пользователя с примерами
- Справочник по ГОСТ стандартам
- API Reference для разработчиков

### Known Issues
- Интеграция с Tekla Structures API требует окончательной реализации в классах:
  - `WeldingAutomation.AutoPlaceWelds()`
  - `DrawingAutomation.AutomaticallyDimension()`
  - `AttributeManager.CompareAssemblies()`

### Future Plans
- Поддержка 3D визуализации сварки
- Интеграция с системами ERP/1C
- Расширенные фильтры поиска в каталогах
- Импорт каталогов из внешних файлов
- Поддержка пользовательских профилей и болтов
- Многоязычный интерфейс
